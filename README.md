# 聚会地点规划工具

一个基于百度地图 JSAPI 的 Web 应用，帮助分布在不同地点的朋友们找到合适的聚会地点。

## 功能特点

- **多方位置标注**：在地图上显示所有人的居住位置
- **智能中间点搜索**：自动计算所有人的地理中心点，搜索附近候选地点
- **自定义搜索关键词**：支持搜索餐厅、酒店、网吧、电影院等各种目的地
- **精细通勤时间显示**：显示地铁通勤时间（分钟/小时）
- **路线可视化**：点击任意候选点，显示所有人到该地点的地铁线路
- **自动排序**：已计算的候选点按最长通勤时间自动排序
- **API 优化**：按需加载路线，避免百度地图 API 限流
- **添加/删除朋友**：动态管理参与聚会的朋友
- **URL 分享**：生成分享链接，好友打开即可加载相同的朋友列表

## 使用方法

1. 访问在线地址：https://zhongsongze.github.io/dinner-plan/
2. 在「添加新朋友」表单中输入名字和地址，点击「添加」
3. 重复步骤2添加所有参与聚会的朋友
4. 在搜索框输入目的地类型（如：餐厅、酒店、网吧）
5. 点击「搜索候选地点」按钮
6. 点击任意候选点查看详细路线和通勤时间
7. 点击「生成分享链接」可生成链接发给朋友

## 分享功能

1. 添加好所有朋友后，点击「生成分享链接」
2. 复制生成的链接
3. 将链接发给朋友，他们打开后会自动加载相同的朋友列表

## 配置说明

如需使用自己的百度地图 AK：

1. 访问 [百度地图开放平台](https://lbsyun.baidu.com/)
2. 创建应用获取 AK（浏览器端）
3. 修改 `index.html` 中的 AK：

```javascript
// 找到这一行，替换为你的 AK
<script src="//api.map.baidu.com/api?v=1.0&type=webgl&ak=你的AK"></script>
```

## 本地开发

```bash
# 启动本地服务器
python3 -m http.server 8080

# 访问 http://localhost:8080/
```

## 注意事项

- 百度地图免费 API 有调用次数限制，请勿频繁搜索
- 搜索结果按通勤时间自动排序
- 最多支持约 10 个候选点的路线计算
